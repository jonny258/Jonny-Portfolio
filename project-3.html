Note-Taker

This application is a simple note-taking app that allows users to create, save, view, and delete notes.

Live Link

Project Overview

Saving Notes:
Users can enter a new note's title and text in the right-hand column of the Notes page. As soon as a new note is entered, a Save icon dynamically appears in the navigation bar at the top of the page, indicating that the note can be saved. Clicking on the Save icon triggers the saving process, which adds the new note to the list of existing notes in the left-hand column. This feature ensures that users can securely save their notes for future reference.

Viewing Notes:
The left-hand column of the Notes page displays a list of existing notes. Users can easily access their previously created notes by clicking on any note in the list. Clicking on a specific note dynamically loads its title and text in the right-hand column, allowing users to conveniently view and edit their notes. This feature enables quick access and seamless viewing of stored notes, facilitating efficient note management.

Creating New Notes:
The Write icon in the navigation bar provides users with a straightforward way to create new notes. Clicking on the Write icon instantly presents users with empty fields to enter a new note's title and text in the right-hand column. Users can begin typing their thoughts or important information right away, streamlining the note creation process. This feature promotes a smooth and intuitive user experience, encouraging users to capture their ideas without any obstacles.

Deleting Notes:
Each note in the list displayed in the left-hand column has a Delete icon next to it. Users can remove unwanted notes from their collection by simply clicking on the Delete icon corresponding to a specific note. Upon deletion, the note is promptly removed from the list, and the right-hand column is cleared to make way for other notes. This feature provides users with control over their note collection, enabling them to easily manage and declutter their notes.
